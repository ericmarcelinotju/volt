# Volt

Vue 3 + Typescript & Tailwind boilerplate for administrator management dashboard application

## Features

- Generic tables and form components
- Uniform page template
- Built-in validation system
- Easy to customize UI
- i18n

## Getting started

Create .env file

```
npm install
npm run dev
```

## Folder structure

- api
- api-mock
- assets
- components
- composables
- config
- pages
- plugins
- router
- stores
- typings
- utils
